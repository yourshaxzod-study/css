CSS Grid Introduction

What is CSS Grid?

CSS Grid Layout is a two-dimensional layout system that allows you to create complex, responsive layouts with rows and columns. Unlike Flexbox which is primarily one-dimensional, Grid gives you complete control over both horizontal and vertical space.

Why Use CSS Grid?

  • Two-dimensional layout control (rows and columns)
  • Easy creation of complex layouts
  • Built-in responsive capabilities
  • Overlapping elements support
  • Better semantic HTML structure
  • Precise item placement

Grid Terminology

  • Grid Container: The parent element with display: grid
  • Grid Items: The direct children of the grid container
  • Grid Lines: The dividing lines that make up the structure
  • Grid Track: The space between two adjacent grid lines
  • Grid Cell: The space between two adjacent row and column lines
  • Grid Area: The total space surrounded by four grid lines

Basic Grid Setup

HTML Structure

<div class="grid-container">
  <div class="grid-item">Item 1</div>
  <div class="grid-item">Item 2</div>
  <div class="grid-item">Item 3</div>
  <div class="grid-item">Item 4</div>
</div>

CSS Setup

.grid-container {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr; /* 3 equal columns */
  grid-template-rows: 100px 100px; /* 2 rows with 100px height */
  gap: 10px; /* Space between items */
}

Common Grid Properties

Container Properties

.grid-container {
  display: grid;
  grid-template-columns: 200px 1fr 100px;
  grid-template-rows: 50px auto 50px;
  grid-template-areas: "header header header"
                         "sidebar main aside"
                         "footer footer footer";
  gap: 10px;
  justify-items: stretch;
  align-items: stretch;
  justify-content: center;
  align-content: center;
}

Item Properties

.grid-item {
  grid-column: 1 / 3; /* Span from column 1 to 3 */
  grid-row: 2 / 4; /* Span from row 2 to 4 */
  grid-area: header; /* Use named area */
  justify-self: center;
  align-self: center;
}

Grid Units

/* fr unit - fraction of available space */
grid-template-columns: 1fr 2fr 1fr; /* 25% 50% 25% */

/* minmax() function */
grid-template-columns: minmax(100px, 1fr) 200px;

/* repeat() function */
grid-template-columns: repeat(3, 1fr);
grid-template-columns: repeat(auto-fit, minmax(200px, 1fr));

/* Mixed units */
grid-template-columns: 200px 1fr 100px;
grid-template-rows: 50px auto 50px;
